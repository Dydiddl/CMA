## DB 연계
- headquarters 테이블
  - id: UUID
  - company_name: VARCHAR(255)
  - business_number: VARCHAR(20)
  - representative_name: VARCHAR(100)
  - address: TEXT
  - phone: VARCHAR(20)
  - established_date: DATE
  - created_at: TIMESTAMP
  - updated_at: TIMESTAMP
