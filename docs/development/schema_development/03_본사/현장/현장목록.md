# 현장 목록 관리

## 현장 정보
- 현장명 <- 공사명, 공사 계약과 같음
- 현장 주소
- 현장 대표번호
- 현장 책임자
- 착공일
- 준공예정일
- 현장 상태

## 현장 상태
- 준비중
- 진행중
- 완료
- 중단

## DB 연계
- sites 테이블
  - id: UUID
  - name: VARCHAR(100)
  - address: TEXT
  - phone: VARCHAR(20)
  - manager_id: UUID
  - start_date: DATE
  - end_date: DATE
  - status: VARCHAR(20)
  - created_at: TIMESTAMP
  - updated_at: TIMESTAMP

## 관련 테이블
- site_workers
- site_equipment
- site_documents
- site_progress
