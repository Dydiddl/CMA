# 공무부 관리

## 기본 정보
- 부서장
- 부서 연락처
- 부서 이메일
- 주요 업무

## 주요 업무
- 공사 관리
  - 공사 계획 수립
  - 공사 진행 관리
  - 공사 완료 관리
- 안전 관리
  - 안전 교육
  - 안전 점검
  - 사고 예방
- 품질 관리
  - 품질 계획
  - 품질 검사
  - 품질 기록
- 자재 관리
  - 자재 계획
  - 자재 조달
  - 자재 보관

## DB 연계
- departments 테이블
  - id: UUID
  - name: VARCHAR(50) = '공무부'
  - type: VARCHAR(20) = 'headquarters'
  - manager_id: UUID (부서장)
  - phone: VARCHAR(20)
  - email: VARCHAR(255)
  - created_at: TIMESTAMP
  - updated_at: TIMESTAMP
