# 계약관리 폴더 안내

## 업무 개요
- 계약관리 업무는 계약의 전 단계(계약전, 계약, 착공, 준공, 청구)를 체계적으로 관리하는 것을 목표로 합니다.
- 각 단계별로 제출서류, 담당자, 업무 흐름을 명확히 하여 실무와 시스템의 일관성을 유지합니다.

## 폴더 구조
- 01_계약전: 계약 체결 전 단계
- 02_계약: 계약 체결 및 관리 단계
- 03_착공: 착공 신고 및 준비 단계
- 04_준공: 준공 신고 및 정산 단계
- 05_청구: 대금 청구 및 지급 단계

## 단계별 주요 내용
- 각 폴더의 README.md 및 제출서류.md 참고

## 관리 기준
- 각 단계별 제출서류 및 이력은 반드시 기록 및 보관
- 업무 담당자 및 검토자 지정 필수
- 변경 이력, 승인 이력 등은 시스템 또는 문서로 관리
